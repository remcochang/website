Recent Publications

NN Architecture Search

We introduce REMAP, a system that helps a user perform guided search of neural network architectures.

(VAST)

 

Effect of Latency

A study of how latency in an interactive system affects a user's search behavior in a visualization interface.

(InfoVis)

 

AutoML + VIS

A visual analytics system (Snowcat) that integrates AutoML for model construction, comparison, and selection.

(EuroVis) (EuroVis Short)

 

Kyrix

An optmized database + visualization system for web-based multi-canvas pan-zoom interfaces. (MIT)

(EuroVis), (CIDR), (VLDB)

 

At a Glance

Modeling the perceived complexity of time-series visualization using "pixel approximate entropy"

(InfoVis)

 

RNNbow

Visualization of the learning rates (the relative gradient contributions) of a recurrent neural network.

(CGA)

 

Weber's Law 2.0!

Modeling the perception of correlation in scatterplots. Follow up work to the Weber's law paper in 2014.

(TVCG)

 

Beagle

Machine learning and classification of visualization types using SVG elements in HTML

(CHI)

 

Medical Decision Making

User study and visualization tool (PROACT) to help prostate cancer patients understand their personal health risk.

(InfoVis)
